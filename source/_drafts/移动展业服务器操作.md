# 移动展业服务器和数据库操作

## 发布

1. 首先从svn上拉下需要发布的版本,一般在02-版本管理>上线版本管理

2. 备份

   -  文件夹：压缩命令 xxx为文件夹名//xxx可以是多文件。

     ```
     tar -cvf xxx日期bk.tar.gz xxx
     ```

   - 文件：在xftp中改名，加上bk日期后缀

3. 将文件传输到对应位置

   - 如果是wap，直接删除原来的文件夹，然后unzip压缩文件名

   - 如果是jar 先

     ```
     ps -ef|grep java
     ```

     查询进程，然后

     ```
     kill -9 xxx
     ```

     杀掉进程，文件传输完成后，重启进程，类似如下命令

     ```
     
     ```
   
4. 查验版本。
